Overview
--------

Create local dev environment to mine ethereum.

Installation / Usage
--------------------

To install use pip:

    $ pip install PyEthereum


Or clone the repo:

    $ git clone https://github.com/cofm21/PyEthereum.git
    $ python setup.py install
